🛒 Amazon Clone (HTML & CSS)
This project is a front-end clone of the Amazon homepage built using HTML and CSS. It replicates the layout and design of Amazon's landing page, including navigation, hero section, and product category cards.

📌 Features
Responsive Navigation Bar
Amazon-style Hero Banner
Product Category Sections
Clean Grid & Flexbox Layout
Modern UI Styling using CSS
Fully built using HTML & CSS only

🛠️ Technologies Used
HTML5
CSS3
Flexbox
CSS Grid

🎯 Purpose of the Project
This project is created to:
Practice HTML & CSS layout design
Improve UI cloning skills
Learn Flexbox and Grid
Understand real-world webpage structure

💡 Future Improvements
Add JavaScript functionality
Implement responsive design for mobile devices
Create product pages
Add search functionality
Connect to backend/database
